# Summary

This is a flexible robot hand.

Many robot hands require assembly... 
But this model can be printed at once and using filament is ABS(not flexible filament).

http://blog.smartrobotics.jp/2016/12/21/3d%e3%83%97%e3%83%aa%e3%83%b3%e3%82%bf%e3%81%a7%e4%b8%80%e4%bd%93%e9%80%a0%e5%bd%a2%e5%8f%af%e8%83%bd%e3%81%aa%e3%83%95%e3%83%ac%e3%82%ad%e3%82%b7%e3%83%96%e3%83%ab%e3%83%8f%e3%83%b3%e3%83%89/

Servo : SG92R
http://www.towerpro.com.tw/product/sg92r-7/

Servo fixing method : M2-8mm 

Thumb string : https://www.amazon.com/dp/B007DLVLDY/ref=cm_sw_r_tw_dp_x_Mmmwyb8TQ6S1N
Other finger string : wire
